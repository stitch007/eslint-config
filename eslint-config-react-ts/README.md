# @stitch007/eslint-config-react-ts

## 使用

安装依赖

```shell
# npm i -D @stitch007/eslint-config-react-ts
# yarn add -D @stitch007/eslint-config-react-ts
pnpm i -D @stitch007/eslint-config-react-ts
```

配置文件

```javascript
// .eslintrc.js
module.exports = {
  extends: ['@stitch007/eslint-config-react-ts'],
}
```
